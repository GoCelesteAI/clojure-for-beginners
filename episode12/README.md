# Episode 12 — Lazy Sequences

Student code for **Clojure for Beginners in Neovim, Episode 12**.

YouTube: https://www.youtube.com/@CelesteAI

## What this episode covers

- **range** — finite `(range 5)` and infinite `(range)`
- **take** — grab N items from any (possibly infinite) sequence
- **repeat** — infinite repetition: `(repeat "hello")`
- **iterate** — build a sequence by applying a function repeatedly
- **take-while** — take items while a condition holds
- **Laziness** — infinite sequences cost nothing until realized

## Run the file

```bash
clj -M lazy.clj
```
